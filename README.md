# 🚀 AI TaskFlow Manager

> An intelligent AI-powered task management platform that transforms natural language into structured, actionable workflows using Large Language Models.

AI TaskFlow Manager helps users organize their work effortlessly by leveraging AI to understand tasks, generate detailed descriptions, prioritize work, break complex goals into manageable subtasks, and provide intelligent assistance through an integrated AI chatbot.

---

## ✨ Features

🧠 **Natural Language Task Creation**
- Create tasks simply by describing them in plain English.

🤖 **AI Task Description Generator**
- Automatically generates detailed and meaningful task descriptions.

⚡ **Smart Priority Prediction**
- Suggests task priority based on urgency and context.

📋 **Automatic Subtask Generation**
- Breaks large tasks into smaller actionable steps.

💬 **AI Productivity Assistant**
- Interactive chatbot that helps users manage, organize, and plan their work efficiently.

📊 **Analytics Dashboard**
- Visual insights into productivity and task completion.

🔒 **Secure User Authentication**
- User registration, login, and personalized dashboards.

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/devarasasank31/AI-TASK-FLOW-MANAGER.git
```

Move into the project

```bash
cd AI-TASK-FLOW-MANAGER
```

Install dependencies

```bash
pip install django==4.2.7 groq==1.2.0
```

Apply migrations

```bash
python manage.py migrate
```

Run the server

```bash
python manage.py runserver
```

Open your browser

```
http://127.0.0.1:8000
```
